vLab: Managing and Provisioning VMs for Labs

Gateway The I/O requests from the QEMU block driver are handled by the Gateway. It is not necessary that all the data is present on the same node, in that case gateway may interact with other nodes also. To read/write the actual data from/to disk Gateway talk to Object Manager,a component of Object Store. Gateway calculates the target nodes based on the consistent hashing algorithm and request is forwarded to the object manager of the target node.
